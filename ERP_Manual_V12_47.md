# 1. 系統概述

## 1.1 系統簡介

八方環球 ERP 系統是一套專為水產批發業設計的報價與庫存管理系統，採用現代化網頁技術開發，支援電腦與手機操作。

## 1.2 核心功能

| 功能模組 | 說明 |
|----------|------|
| 報價管理 | 快速建立報價單、支援多價格等級 |
| 庫存查詢 | 即時庫存查詢、多倉庫管理 |
| 客戶管理 | 客戶資料、歷史報價記錄 |
| 低庫存提醒 | 自動檢測庫存不足、安全庫存設定 |
| 雲端同步 | Firebase 即時同步、多裝置存取 |
| LINE 分享 | 一鍵分享報價單到 LINE |

## 1.3 系統需求

- **瀏覽器**：Chrome、Safari、Edge（建議最新版本）
- **網路**：需要網路連線（雲端同步）
- **裝置**：電腦、平板、手機皆可使用

# 2. 快速開始

## 2.1 登入系統

1. 開啟系統網址或 HTML 檔案
2. 輸入密碼（請洽管理員取得）
3. 登入後自動連線 Firebase 雲端資料庫

> 💡 系統會記住登入狀態 7 天，期間不需重複登入

## 2.2 帳號權限

| 角色 | 權限說明 |
|------|----------|
| **管理員 (admin)** | 完整權限，可進入系統管理頁面 |
| **業務 (sales)** | 報價與查詢權限，無法進入管理頁面 |

## 2.3 主要介面

| 首頁 | 系統總覽、快速操作入口、統計資訊 |

| 歷史 | 查看歷史報價記錄、重新載入報價 |

# 3. 功能模組說明

## 3.1 報價功能

### 建立報價單步驟

1. 點擊「報價」頁籤
2. 點擊「+ 選擇產品」按鈕
3. 搜尋或瀏覽產品清單
4. 選擇產品後設定數量
5. 調整價格（可選擇不同價格等級）
6. 填寫客戶資訊
7. 點擊「儲存」或「預覽」

### 價格等級

| 等級 | 說明 |
|------|------|
| 門市售價 | 一般零售價格 |
| 會員95% | 95折優惠價 |
| 餐廳90% | 9折優惠價 |
| 生意價 | 標準批發價格 |
| 整件價 | 整件批發價 |
| VIP價 | VIP專屬價格 |
| 業務底價 | 業務人員底價 |

### 客戶資訊（左側欄位）

| 欄位 | 說明 |
|------|------|
| 客戶名稱 | 客戶公司或個人名稱 |
| 價格等級 | 下拉選擇價格等級 |
| 付款方式 | 月結30天/月結60天/現金/貨到付款/匯款/支票 |
| 電話/手機 | 聯絡電話 |
| 地址 | 送貨地址 |
| 日期 | 報價單日期 |
| 預覽報價單 | 預覽/列印報價單 |

## 3.2 LINE 分享功能

### 訊息格式

```
【2024-12-17 報價單】
王大明水產行｜月結30天

1. 特級白蝦 40/50
   40/50 規格 / 淨重1.2kg / 厄瓜多爾
   $350/件

2. 草蝦 20/30
   20/30 規格 / 淨重1kg
   $280/包

3. 鮭魚切片
   厚切 / 每片約200g / 冷凍
   $180/片
```

### 格式說明

| 區塊 | 內容 |
|------|------|
| **標題** | 【日期 報價單】 |
| **客戶資訊** | 客戶名稱｜結帳方式 |
| **產品明細** | 序號、品名、規格、單價/單位 |
| **備註** | 有才顯示 |

> 💡 **設計理念**：報價階段只需讓客戶知道「有什麼產品」和「單價多少」，成交後以 ERP 銷貨單為準。

## 3.3 歷史報價

- 查看所有歷史報價記錄
- 依日期排序（最新在前）
- 可載入、刪除、LINE分享

# 4. 低庫存提醒功能

## 4.1 功能說明

系統會自動檢測主要倉庫的庫存，當產品總庫存低於安全庫存時，透過紅點鈴鐺提醒。

| 項目 | 說明 |
|------|------|
| 檢查時機 | 每次進入系統時自動檢查 |
| 檢查範圍 | 所有主要出貨倉庫的庫存加總 |
| 觸發條件 | 總庫存 < 安全庫存 |
| 預設安全庫存 | 40 件（可個別調整） |

## 4.2 紅點鈴鐺提醒

### 位置
- 上方固定標題列
- 設定圖示（⚙️）左邊
- 所有頁面都會顯示

### 顯示說明
| 狀態 | 顯示 |
|------|------|
| 無低庫存 | 僅顯示鈴鐺圖示 |
| 有低庫存 | 鈴鐺 + 紅點數字（閃爍動畫） |
| 超過99項 | 顯示「99+」 |

## 4.3 主要監控倉庫

| 倉庫名稱 | 類型 |
|----------|------|
| 崇文_白蝦 | 白蝦成品 |
| 八方_白蝦 | 白蝦成品 |
| 崇文_成品 | 其他成品 |
| 八方_成品 | 其他成品 |
| 崇文_原料 | 原料庫存 |
| 八方_原料 | 原料庫存 |
| 崇文_半成品庫 | 半成品 |

## 4.4 低庫存彈窗設計（V12.45 新版）

### 設計特色

| 設計要點 | 說明 |
|----------|------|
| **卡片式佈局** | 每個產品獨立卡片，左側色條快速辨識 |
| **視覺警示** | 🔴 紅色 = 緊急（≤10）、🟠 橘色 = 警告（>10） |
| **儀表板數字** | 三欄並排：缺貨數、安全庫存、目前總量 |
| **折疊細節** | 「查看倉庫分佈」預設隱藏，點擊展開 |
| **手機優化** | 底部彈出、上方圓角、90vh 高度 |

### 字體層級

| 元素 | 樣式 |
|------|------|
| 產品名稱 | 最大、最黑 |
| 規格 | 變小、變灰 |
| 品號 | 等寬小字 |

## 4.5 產品類型與處理建議

| 產品類型 | 其他倉庫有庫存 | 處理建議 |
|----------|----------------|----------|
| 採購品 | ✅ 有 | 🔵 需調撥 |
| 採購品 | ❌ 無 | 🟢 需採購 |
| 自製品 | ✅ 有 | 🔵 需調撥 |
| 自製品 | ❌ 無 | 🟣 需生產 |

## 4.6 安全庫存設定

- **預設值**：40 件
- **個別設定**：可針對特定產品調整
- **設為 0**：該產品不觸發警報

# 7. 常見問題

## Q1: 安全庫存設定為 0 有什麼效果？
該產品將不會出現在低庫存警報中，適合客戶訂製品或臨時寄放商品。

## Q2: LINE 分享為什麼沒有總金額？
報價階段主要讓客戶了解產品和單價，成交後以 ERP 銷貨單為準。

## Q3: 如何區分採購品和自製品？
在報價表 Excel 中新增「類型」欄位，填入「採購品」或「自製品」。

## Q4: 鈴鐺沒有顯示紅點？
可能原因：無低庫存產品、資料未載入、安全庫存都設為 0。
